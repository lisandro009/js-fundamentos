JSON Javascript object notation o notacion de objeto Javascript

Antes de json se usaba xml

JSON json mdn

Es un formato ligero de intercambio de datos

JS tiene JSON que es una interfaz que nos ayuda a manejar datos json

Tiene dos metodos

  • Parse: convierte datos json a datos validos para js (para recibir), para poder ser utilizados en la logica de programacion
  • stringify: convierte del lenguaje JS a texto (para mandar)
